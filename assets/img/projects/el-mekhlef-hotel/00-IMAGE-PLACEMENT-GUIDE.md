# Image Slots — El Mekhlef Hotel

**لا يوجد اسم إجباري لأي صورة.** ضع صورة واحدة بأي اسم داخل فولدر المكان الصحيح.

- Project page: `projects/el-mekhlef-hotel/index.html`
- Project folder: `assets/img/projects/el-mekhlef-hotel/`

| Folder | المكان في الموقع | Required |
|---|---|---|
| `01-project-card-image/` | تظهر في كارت المشروع داخل الصفحة الرئيسية، صفحة الأعمال، وأقسام المشاريع المرتبطة. | نعم |
| `02-project-page-main-hero/` | تظهر كأول صورة كبيرة في أعلى صفحة تفاصيل المشروع. | نعم |
| `03-project-page-gallery-image-01/` | تظهر في أول مكان داخل معرض صور صفحة المشروع. | نعم |
| `04-project-page-gallery-image-02/` | تظهر في ثاني مكان داخل معرض صور صفحة المشروع. | نعم |
| `05-project-page-gallery-image-03/` | تظهر في ثالث مكان داخل معرض صور صفحة المشروع. | نعم |
| `06-project-page-gallery-image-04/` | تظهر في رابع مكان داخل معرض صور صفحة المشروع. | نعم |
| `07-project-social-share-image-optional/` | تستخدم كصورة المعاينة عند مشاركة رابط صفحة المشروع على واتساب وفيسبوك ولينكدإن. | اختياري |

## ملاحظات
- يمكن أن يكون اسم الصورة عربيًا أو إنجليزيًا أو رقمًا؛ لا توجد تسمية إلزامية.
- ضع صورة واحدة فقط داخل كل فولدر حتى يكون الربط واضحًا.
- يمكنك استخدام نفس الصورة في أكثر من فولدر عندما تريد تكرارها في أكثر من مكان.
- عند إرسال الصور لاحقًا سيتم قراءة الاسم الفعلي داخل كل فولدر وربطه بالمكان المحدد.
